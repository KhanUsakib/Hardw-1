Vanilla javscript

JavaScript is a versatile programming language for web development. It runs in browsers to make websites interactive and dynamic. With Node.js, it also works on servers. Used for frontend/backend, mobile apps, and even desktop applications.

React

React is a JavaScript library for building user interfaces, especially single-page applications. Created by Facebook, it uses reusable components and virtual DOM for fast rendering. Popular for modern web apps with interactive UIs.

Nodejs

Node.js is a runtime environment that allows JavaScript to run on servers. It uses an event-driven, non-blocking I/O model making it efficient for real-time applications. Perfect for building scalable backend services and APIs.

MongobBD

MongoDB is a NoSQL database that stores data in flexible, JSON-like documents. It's schema-less, scalable, and handles large volumes of data efficiently. Great for modern applications with evolving data structures.
